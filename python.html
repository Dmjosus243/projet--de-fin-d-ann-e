Le langage de programmation Python est un langage de programmation qui est connue pour sa syntaxe claire et lisible , il a été publié pour la première fois en 1989 par Guido van Rossum ;c'est un langage de programmation utilisé dans plusieurs domaines tels que dans le domaine de la cybersécurité , programmation web

C'est un langage de programmation utilisé par des grandes entreprises tels que : Google utilise Python pour l'intelligence artificielle, l'analyse de données et le développement web;Instagram l'utilse pour son backend qui se repose largement sur Python pour gérer des millions d'utilisateurs; Dropbox qui a adopté Python pour son infrastructure cloud et a même embauché Guido van Rossum qui le créateur de Python. ;Netflix exploite Python pour l'analyse de données et la personnalisation des recommandations de sa clientèle;NASA utilise Python pour le traitement des données scientifiques et les simulations;BitTorrent a intégré Python dans son protocole de partage de fichiers;Industrial Light & Magic utilise aussi Python pour les effets spéciaux dans les films.

Fonction Description Exemple
print() Affiche du texte à l'écran print("Hello World")
input() Lit une entrée utilisateur nom = input("Entrez votre nom : ")
len() Renvoie la longueur d'une chaîne de caractères longueur = len("Bonjour")
type() Renvoie le type d'une variable type_variable = type(42)
int() Convertit une valeur en entier nombre = int("10")
float() Convertit une valeur en nombre à virgule flottante nombre_flottant = float("3.14")
str() Convertit une valeur en chaîne de caractères chaine = str(42)
list() Crée une liste à partir d'une séquence ma_liste = list(range(5))
dict() Crée un dictionnaire à partir de paires clé-valeur mon_dict = dict(a=1, b=2)
set() Crée un ensemble à partir d'une séquence mon_set = set([1, 2, 3])
range() Crée une séquence de nombres ma_sequence = range(10)
sum() Renvoie la somme des éléments d'une séquence total = sum([1, 2, 3])
max() Renvoie la valeur maximale d'une séquence maximum = max([1, 2, 3])
min() Renvoie la valeur minimale d'une séquence minimum = min([1, 2, 3])
sorted() Renvoie une liste triée liste_triee = sorted([3, 1, 2])
enumerate() Renvoie un itérateur qui produit des paires (index, valeur) for index, valeur in enumerate(["a", "b", "c"]):
zip() Regroupe des éléments de plusieurs séquences for a, b in zip([1, 2], ["x", "y"]):
map() Applique une fonction à chaque élément d'une séquence resultat = list(map(lambda x: x * 2, [1, 2, 3]))
filter() Filtre les éléments d'une séquence selon une condition resultat = list(filter(lambda x: x > 0, [-1, 0, 1, 2]))
