Bienvenue dans notre documantation CSS

Dans cette section, nous allons explorer les bases du CSS, comment l'utiliser pour styliser vos pages web et quelques astuces pour améliorer votre design.

Historique

Ce langage de dévloppement web a été crée par Håkon Wium Lie et Bert Bos; Håkon Wium Lie proposa pour la première fois le concept de CSS en 1994,alors qu'il travaillait avec Tim Berners-Lee au (CERN).Par la suite ,Bert Bos a également apporté des contributions importantes ,devenant co-auteur de la première recpmmandation CSS (CSS1) en 1996.Auparavant le développement des différentes parties de la spécification CSS fut realisé de façon synchrone ,permettant d'avoir une version pour l'ensemble de la recommandation; ce pour cela vous avez peut- être entendu parlé du CSS1,CSS2.1 voire CSS3

Qu'est-ce que le CSS ?

Le CSS acronyme de Cascading Style Sheets est un langage de programmation fondamental dans la programmation web avec l'HTML et le JavaScript.Son rôle principal est de décrire la présentation des documents écrits en HTML;cést-à-dire que le CSS permet de de définir l'apparence visuele d'un site web , en séparant cette présentation du contenu structurel .

Le CSS permet de contrôler la mise en page, les couleurs, les polices, les marges, les espacements et bien d'autres aspects visuels d'une page web. Il est utilisé pour rendre les sites web attrayants et faciles à naviguer.

Qu'est-ce que est réellement le CSS ?

De la même façon que l'HTML , le CSS n'est pas vraiment un langage de programmation.C'est un langage de feuille de style,cela veut-dire qu'il permet de donner de styles sur différents éléments sélectionnés dans un document HTML .Par exemple,on peut sélectionner des éléments HTML qui sont tous des paragraphes et afficher le texte dans cette balise en couleur rouge en code CSS : p {

color: red;
}

Ceci est un paragraphe affiché en rouge grâce au CSS.Où on vient de sélectionner juste une balise

Selection de plusieurs balises

Il est aussi possible de sélectioner plusieurs balises en même temps ,pour une stylisation collective , il suffit de placer plusieurs sélecteurs et de les séparés avec des virgules.Par exemple : p , li, h1, {

color: blue;
} Là comme vous pouvez le voir nous avons sélectionner plusieurs sélecteurs , puis nous les avons donnés comme couleur commune "blue"
