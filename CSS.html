Bienvenue dans notre documantation CSS

Dans cette section, nous allons explorer les bases du CSS, comment l'utiliser pour styliser vos pages web et quelques astuces pour améliorer votre design.

Historique

Ce langage de dévloppement web a été crée par Håkon Wium Lie et Bert Bos; Håkon Wium Lie proposa pour la première fois le concept de CSS en 1994,alors qu'il travaillait avec Tim Berners-Lee au (CERN).Par la suite ,Bert Bos a également apporté des contributions importantes ,devenant co-auteur de la première recpmmandation CSS (CSS1) en 1996.Auparavant le développement des différentes parties de la spécification CSS fut realisé de façon synchrone ,permettant d'avoir une version pour l'ensemble de la recommandation; ce pour cela vous avez peut- être entendu parlé du CSS1,CSS2.1 voire CSS3

Qu'est-ce que le CSS ?

Le CSS acronyme de Cascading Style Sheets est un langage de programmation fondamental dans la programmation web avec l'HTML et le JavaScript.Son rôle principal est de décrire la présentation des documents écrits en HTML;cést-à-dire que le CSS permet de de définir l'apparence visuele d'un site web , en séparant cette présentation du contenu structurel .

Le CSS permet de contrôler la mise en page, les couleurs, les polices, les marges, les espacements et bien d'autres aspects visuels d'une page web. Il est utilisé pour rendre les sites web attrayants et faciles à naviguer.

Qu'est-ce que est réellement le CSS ?

De la même façon que l'HTML , le CSS n'est pas vraiment un langage de programmation.C'est un langage de feuille de style,cela veut-dire qu'il permet de donner de styles sur différents éléments sélectionnés dans un document HTML .Par exemple,on peut sélectionner des éléments HTML qui sont tous des paragraphes et afficher le texte dans cette balise en couleur rouge en code CSS : p {

color: red;
}

Ceci est un paragraphe affiché en rouge grâce au CSS.Où on vient de sélectionner juste une balise

Selection de plusieurs balises

Il est aussi possible de sélectioner plusieurs balises en même temps ,pour une stylisation collective , il suffit de placer plusieurs sélecteurs et de les séparés avec des virgules.Par exemple : p , li, h1, {

color: blue;
} Là comme vous pouvez le voir nous avons sélectionner plusieurs sélecteurs , puis nous les avons donnés comme couleur commune "blue"

Les différents types de sélecteurs

Il y a différent types de sélecteurs ci-haut comme vous pouvez le constater , grâce à cela nous voyons que les sélecteurs nous permettent de sélectionner des éléments HTML d'un type de donné dans un document HTML .Mais nous pouvons aussi faire des sélections spécifiques,voici certains types de sélecteurs qui sont les plus utilisés :

Nom du sélecteur Element sélectionné Exemple
Sélecteur de balise Tous les éléments HTML d'un type donné p pour sélectionner tous les <p>
Sélecteur d'ID Il sélectionne tous les éléments HTML avec une "id=" #my-id sélectionne <p id="my-id">
Sélecteur de classe Les éléments d'une page HTML qui ont une "class= " donné .my-class sélectionne <p class="my-class"> et < a class="my-class!"> .
Sélecteur d'attributs Il sélectionne un élément possédant un attribut donné . img[src] sélection <img src=monimage.png"> mais pas <img>.
Sélecteur de pseudo-classe Il sélectionne un élément en fonction de son état ou de sa position dans le document. a:hover sélectionne un lien lorsque l'utilisateur passe la souris dessus.
Sélecteur de pseudo-élément Il sélectionne une partie spécifique d'un élément. p::first-line sélectionne la première ligne d'un paragraphe.
Sélecteur universel Il sélectionne tous les éléments d'un document. * sélectionne tous les éléments.

Pour que tout cela puisse fonctionner il faut tout d'abord appliquer le CSS à un document HTML ,sinon le CSS n'affectera pas la mise en forme de la page web dans le navigateur

Comment appliquer le CSS à un document HTML ?

  1. Ouvrez votre fichier index.html et copiez votre lien vers votre CSS quelques part dans la balise <head> ,puis sauvegardez index.html puis vous actualisez votre page web dans votre navigateur par défaut cela affichera une stylisation de votre page web Si le paragraphe s'affiche en rouge c'est que le CSS a bien été appliqué .
  2. Vous pouvez aussi ajouter le CSS directement dans la balise <style> dans la balise <head> de votre document HTML .

Les polices et les textes

Maintenat que nous avons vus quelques bases de la stylisation CSS , ajoutons quelques règles et informations à notre fichier HTML style.css . Tout d'abord améliorons notre police et le texte

  1. Pour commencer à changer le modéle de texte que vous voulez on déclare d'abord le sélecteur font-family qui répresente les polices qu'on veut utiliser pour les textes
  2. Pour la taille de la police nous avons le sélecteur font-size qui nous permet de définir la taille que nous voulons pour notre texte ,il faut savoir que la granndeur est définie en pixels

Ces sélecteurs de base définnise une police différente que la police par défaut et de la taille de base pour toute la page

voici un exemple de code CSS pour changer la police et la taille du texte dans votre fichier style.css :
body {

font-family: Arial, sans-serif; /* Change la police de la page */

font-size: 16px; /* Définit la taille de la police */
} Tout ce qui entre les accolades {} est le code CSS qui va s'appliquer à la balise <body> de votre document HTML . Et tout ce qui est entre /* et * / est un commentaire qui n'est pas pris en compte par le navigateur mais qui peut être utile pour expliquer le code ou pour laisser des notes pour vous-même ou pour d'autres développeurs. Ensuite , fixons les tailles des différents textes contenus dans le corps du HTML .Nous allons également centrer le texte du titre et donner une taille de ligne et un espacement de caractère entre les lettres pour le contenu du corps (body) du document soit plus lisible ; voici un exemple de cela :
h1 {

text-align: center; /* Centre le titre */

font-size: 20px; /* Définit la taille du titre */
}

p {

line-height: 1.5; /* Définit l'espacement entre les lignes */

letter-spacing: 1px; /* Définit l'espacement entre les lettres */
} dans cet exemple nous avons utilisé le sélecteur h1 pour styliser le titre de la page et le sélecteur p pour styliser les paragraphes du corps du document HTML .Nous avons aussi utilisé le séleteur text-align pour centrer le titre de la page et le sélecteur font-size pour définir la taille de la police du texte du titre .Nous avons aussi utilisé le sélecteur line-height pour définir l'espacement entre les lignes du texte et le sélecteur letter-spacing pour définir l'espacement entre les lettres du texte .Vous pouvez ajuster ces valeurs en px (pixels) ou en em (unités relatives à le taille de la police ); comme vous le voulez pour votre page web.Cela voous donnera un affichage comme ceci :

Les boîtes

Dans le CSS comme vous le voyez tout tourne majoritairement autour des boîtes : pour définir leurs tailles , leurs couleurs, leurs positions etc.Les éléments HTML d'une page web ,peuvent pour la plupart , être vus comme des boîtes superposées les unes sur les auters .

C'est pour cela que l'architecture du CSS ;est principalement basée sur une architecture des boîtes.Chacun de ces blocs sélecteurs prend un certain esoace sur la page , de cette façon nous avons :

  1. padding : est utilisé pour modifier l'espace autour , proche du contenu ;par exemple l'espace autour du texte d'un paragraphe .
  2. border : est utilisé pour modifier la ligne qui est juste autour du padding ( en français cela correspond à la bordure)
  3. margin : est utilisé pour modofier l'espace extérieur , autour d'un élément( en farnçais cela correspond à la marge)
Dans cette section nous avons ,nous utilisons aussi :
  1. width : pour la modification de la largeur de l'élément
  2. background-color : pour la modification de la couleur derrière le contenu de l'élément et son padding
  3. color : pour choisir la couleur du contenu de l'élément ,généralement du texte.
  4. text-shadow : affiche une ombre portée sur le texte à l'intérieur d'un élément
  5. display : qui définit le mode d'affichage de la page sur une interface donnnée

Allons-y : ajoutons un peu plus de CSS à notre page web ! Continuons à ajouter des sélecteurs à notre fichier HTML à la suite des autres sélecteurs précédent .Il ne faut pas avoir peur de s'exercer des temps en temps , cela vous permettras d'expérimenter et de tester différentes valeurs pour voir ce qu'elles font .

Changer la couleur de la page

Cette règle permet de définir la couleur de l'arriére plan de tout la page web .


body {

background-color: #f0f0f0; /* Change la couleur de fond de la page */
}

Mettre en forme le corps de la page

Nous allons maintenant ajouter des styles pour le corps de la page , en utilisant les sélecteurs padding, border, margin, width, background-color, color, text-shadow et display. Voici un exemple de code CSS pour cela :


body {

padding: 20px; /* Ajoute de l'espace autour du contenu */

border: 1px solid #ccc; /* Ajoute une bordure autour du corps */

margin: 0 auto; /* Centre le corps de la page */

width: 80%; /* Définit la largeur du corps de la page */

background-color: #ffffff; /* Change la couleur de fond du corps */

color: #333333; /* Change la couleur du texte dans le corps */

text-shadow: 1px 1px 2px rgba(0, 0, 0, 0.1); /* Ajoute une ombre portée au texte */

display: block; /* Définit le mode d'affichage du corps */
}

Positionner le titre et le mettre en forme

Nous allons maintenant styliser notre titre et le donner une forme


h1 {

text-align: center; /* Centre le titre */

font-size: 24px; /* Définit la taille du titre */

color: #2c3e50; /* Change la couleur du titre */

margin-bottom: 20px; /* Ajoute de l'espace en dessous du titre */
}

Centrer l'image

Nous allons aussi center notre image


img {

display: block; /* Définit l'image comme un bloc */

margin: 0 auto; /* Centre l'image horizontalement */
}
